# Folder: src/adapters/

> Trigger: Update when supported models or adapter interface changes.

Unified interface for interacting with different generative AI platforms.
[IN] Task Data -> [OUT] Model Interaction, Status Checks.
Invariant: All adapters must implement `BaseAdapter`.

## Core Files
- base-adapter.ts: Abstract base class defining the adapter interface.
- chatgpt-adapter.ts: Implementation for ChatGPT (OpenAI).
- gemini-adapter.ts: Implementation for Gemini (Google).
- adapter-factory.ts: Factory to instantiate the correct adapter based on site type.
