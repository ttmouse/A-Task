# 使用指南页面调试步骤

## 🔍 请按以下步骤检查

### 1. 重新加载扩展

在 `chrome://extensions/` 页面：
1. 找到 A-Task 扩展
2. 点击"刷新"按钮 🔄（或者先移除再重新加载）

### 2. 打开使用指南页面

点击侧边栏的"📖 使用指南"按钮

### 3. 打开浏览器开发者工具

在打开的使用指南页面上：
- **方法1**: 按 `F12` 键
- **方法2**: 右键点击页面 → 选择"检查"
- **方法3**: 按 `Ctrl+Shift+I` (Windows) 或 `Cmd+Option+I` (Mac)

### 4. 查看控制台 (Console)

在开发者工具中：
1. 点击"Console"（控制台）标签
2. 查看是否有红色的错误信息
3. 截图或复制错误信息发给我

### 5. 预期看到的日志

如果一切正常，控制台应该显示：
```
[SidePanel] 初始化使用指南页面
[SidePanel] 使用指南加载成功
```

如果有错误，可能会看到：
```
[SidePanel] 加载使用指南失败: ...
或
[MarkdownRenderer] 加载文件失败: ...
```

### 6. 检查网络请求 (Network)

在开发者工具中：
1. 点击"Network"（网络）标签
2. 刷新页面 (F5)
3. 查看是否有失败的请求（红色）
4. 特别注意：
   - `markdown-renderer.js` 是否加载成功？
   - `guide.md` 是否加载成功？
