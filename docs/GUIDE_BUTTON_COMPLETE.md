## 🔄 如何测试

### 1. 重新加载扩展

在 `chrome://extensions/` 页面：
1. 找到 A-Task 扩展
2. 点击"刷新"按钮（或禁用后重新启用）

### 2. 打开侧边栏

- 点击浏览器工具栏中的 A-Task 图标
- 侧边栏会自动打开

### 3. 查看使用指南

- 在侧边栏顶部找到"📖 使用指南"按钮
- 点击按钮
- 会在新标签页打开使用指南

## ⚠️ 注意事项

### 路径说明

使用指南页面路径：
- **源文件**：`src/pages/sidepanel/sidepanel.html`
- **编译后**：`dist/pages/sidepanel/sidepanel.html`
- **访问方式**：`chrome-extension://[扩展ID]/pages/sidepanel/sidepanel.html`

Markdown 文件路径：
- **源文件**：`src/markdown/guide.md`
- **复制后**：`dist/markdown/guide.md`

### 如果看不到按钮

如果刷新后仍然看不到按钮，请检查：

1. **确认构建成功**
   ```bash
   npm run build
   ```

2. **检查文件是否复制**
   ```bash
   ls -la dist/sidepanel/
   # 应该能看到更新时间是最新的 sidepanel.js
   ```

3. **检查浏览器控制台**
   - 右键点击侧边栏 → 检查
   - 查看是否有 JavaScript 错误

4. **完全重新加载扩展**
   - 在 `chrome://extensions/` 中
   - 先点击"移除"
   - 然后重新加载 `dist/` 目录
